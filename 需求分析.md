# Trios - 产品需求文档

**版本**：v1.0 | **更新日期**：2026-05-09

------

## 一、产品概述

| 项目     | 内容                                            |
| :------- | :---------------------------------------------- |
| 产品名称 | Trios                                           |
| 定位     | 个人生活管理中心                                |
| 核心功能 | 灵感笔记 + 电影清单 + 任务清单                  |
| 运行平台 | iPhone 主屏幕（PWA）                            |
| 技术方案 | 前端 JavaScript + localStorage 本地存储，无后端 |

------

## 二、全局要求

| 要求     | 说明                                     |
| :------- | :--------------------------------------- |
| Tab 切换 | 三个独立模块（灵感/电影/任务），点击切换 |
| 本地存储 | 所有数据自动保存，刷新页面不丢失         |
| 离线可用 | 首次加载后，无网络仍可使用               |
| PWA 安装 | 可添加到 iPhone 主屏幕，全屏无地址栏     |
| 唯一 ID  | 每条数据使用时间戳 + 随机数作为唯一标识  |
| 时间戳   | 每条数据记录创建时间和更新时间           |

------

## 三、模块一：灵感笔记

### 功能清单

| 功能                   | 优先级 | 状态   |
| :--------------------- | :----- | :----- |
| 添加灵感               | P0     | 待开发 |
| 删除灵感               | P0     | 待开发 |
| 列表展示（按时间倒序） | P0     | 待开发 |
| 编辑灵感               | P1     | 待开发 |
| 关键字检索             | P1     | 待开发 |

### 数据结构

javascript

```
{
    id: "唯一标识",
    content: "灵感内容",
    createdAt: "创建时间",
    updatedAt: "更新时间"
}
```



### 检索规则

- 检索范围：`content` 字段
- 匹配规则：不区分大小写，包含关键字即命中
- 无结果时显示“未找到相关灵感”

### 界面元素

- 输入框 + “记录”按钮
- 搜索框 + “清除”按钮

## 四、模块二：电影清单

### 功能清单

| 功能                    | 优先级 | 状态   |
| :---------------------- | :----- | :----- |
| 添加电影                | P0     | 待开发 |
| 删除电影                | P0     | 待开发 |
| 列表展示（按时间倒序）  | P0     | 待开发 |
| 标记看过                | P0     | 待开发 |
| 添加评论（支持多条）    | P0     | 待开发 |
| 评分（十分制 0.0-10.0） | P0     | 待开发 |
| 平均评分显示            | P0     | 待开发 |
| 海报显示（TMDB API）    | P1     | 待开发 |
| 编辑电影名              | P1     | 待开发 |
| 关键字检索              | P1     | 待开发 |
| 观影日期记录            | P1     | 待开发 |

### 数据结构

javascript

```
{
    id: "唯一标识",
    name: "电影名称",
    addedAt: "添加时间",
    watched: true/false,
    watchedAt: "看过时间（首次标记时记录）",
    comments: [
        {
            id: "评论唯一标识",
            text: "评论内容",
            rating: 8.5,        // 十分制，0.0-10.0，保留一位小数
            createdAt: "评论时间"
        }
    ],
    poster: "海报URL"           // P1功能，暂不实现
}
```



### 评分规则

- 范围：0.0 - 10.0
- 精度：保留一位小数（如 7.5、8.8、9.0）
- 与豆瓣、IMDb 一致

### 平均评分计算

- 取所有评论的 rating 平均值
- 保留一位小数
- 无评论时不显示

### 检索规则

- 检索范围：`name` 字段 + `comments[].text` 字段
- 匹配规则：不区分大小写，包含关键字即命中
- 无结果时显示“未找到相关电影”

### 界面元素

- 输入框 + “添加”按钮
- 搜索框 + “清除”按钮
- 列表区域，每条电影显示：名称、平均评分、看过按钮、评论按钮、删除按钮
- 点击评论按钮弹出弹窗：显示已有评论列表 + 添加评论表单（评分下拉选择 + 评论输入框）

## 五、模块三：任务清单

### 功能清单

| 功能                   | 优先级 | 状态   |
| :--------------------- | :----- | :----- |
| 添加任务               | P0     | 待开发 |
| 删除任务               | P0     | 待开发 |
| 完成任务（勾选）       | P0     | 待开发 |
| 优先级（高/中/低）     | P0     | 待开发 |
| 列表展示（按时间倒序） | P0     | 待开发 |
| 关键字检索             | P1     | 待开发 |
| 分类/清单              | P1     | 待开发 |
| 子任务                 | P1     | 待开发 |
| 重复周期               | P1     | 待开发 |
| 拖拽排序               | P1     | 待开发 |
| 截止日期               | P2     | 待开发 |
| 提醒通知               | P2     | 待开发 |

### 数据结构

javascript

```
{
    id: "唯一标识",
    text: "任务内容",
    priority: 1/2/3,        // 1=高，2=中，3=低
    completed: true/false,
    createdAt: "创建时间",
    completedAt: "完成时间",
    updatedAt: "更新时间",
    category: "分类名称",    // P1功能，默认"默认"
    subtasks: [              // P1功能
        {
            id: "子任务唯一标识",
            text: "子任务内容",
            completed: true/false
        }
    ],
    repeat: {                // P1功能
        enabled: true/false,
        type: "daily/weekly/monthly",
        interval: 1,
        endDate: "截止日期"
    },
    dueDate: "截止日期"      // P2功能
}
```



### 优先级映射

| 优先级 | 数值 | 颜色    | 图标 |
| :----- | :--- | :------ | :--- |
| 高     | 1    | #ff3b30 | 🔴    |
| 中     | 2    | #ffcc00 | 🟡    |
| 低     | 3    | #34c759 | 🟢    |

### 检索规则

- 检索范围：`text` 字段 + `subtasks[].text` 字段
- 匹配规则：不区分大小写，包含关键字即命中
- 无结果时显示“未找到相关任务”
- 搜索时不过滤完成状态

### 界面元素

- 输入框 + 优先级下拉菜单 + “添加”按钮
- 搜索框 + “清除”按钮
- 列表区域，每条任务显示：复选框、优先级图标、任务内容、删除按钮
- 已完成任务文字划线
